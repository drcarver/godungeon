Go Dungeon

Go Dungeon is a modern virtual table top using the latest web architectures and methods to provide a environment for players using the D20 and Pathfinder SRD to provide a compationable environment.

Go dungeon clients will be free for players.

Go Dungeon is currently in private beta. We will annonunce a public beta in a few months.
